# 🏆 JavaFX Quiz Game

## 📝 Description
A dynamic Quiz application developed in **Java** using **Object-Oriented Programming**. The game features multiple categories (Sport, Cinema, Manga, etc.) and tracks scores using a **MySQL** database.

The application includes a **smart difficulty algorithm**: the game's difficulty increases progressively as the player answers correctly, ensuring a challenging and engaging experience.

## 🛠️ Features
* **OOP Architecture:** Clean and maintainable code.
* **Progressive Difficulty Algorithm:** Adaptive gameplay that increases challenge based on player performance.
* **Admin & Player Roles:** Separate interfaces for playing and managing the question database (CRUD).
* **Database Integration:** Real-time data retrieval from MySQL.
* **Interactive UI:** created with Scene Builder.

## ⚙️ How to Install
1. **Database:** Import `quiz_db.sql` into your local MySQL server.
2. **Setup:** Open the project in your favorite IDE (IntelliJ or Eclipse).
3. **Run:** Execute the `Main.java` file to start the game.

## 🚀 Technologies
* **Language:** Java (JDK 25)
* **Framework:** JavaFX
* **Database:** MySQL
* **Build Tool:** Maven
